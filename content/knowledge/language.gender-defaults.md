# 语言性别规则：女在前与默认女性

## 女在前原则 {#female-first-ordering}

所有同时指代两性的词语，**女性字在前**：

| 禁用（男在前） | 正确（女在前） |
|-------------|-------------|
| 夫妻 / 夫妇 | 妇夫 |
| 男女 | 女男 |
| 父母 | 母父 |
| 公婆 | 婆翁 |
| 兄弟姐妹 | 姐妹兄弟 |
| 雄雌 | 雌雄 |
| 公母 | 母公 |

「妻」字本身亦为禁词，含男性优先语义。正室称「夫人」，并称用「妇夫」。

## 职业与身份词默认女性 {#occupation-gender-default}

未标注性别的身份与职业词默认指女性：
- 少年：默认指女性青年
- 少郎：男性青年专用
- 老板、掌柜、大夫（官职之外）等：均默认女性

民间医生禁称「大夫」（有官阶含义）；应使用**医师 / 医者 / 先生**。

## 性别化美德词的重新归属 {#gendered-virtue-terms}

雄伟、勇猛、雄姿英发等词形容**女性**。

贤良淑德、贤内助、温柔贤淑等词形容**男性**正室或入赘者。

## 配偶称谓 {#spouse-address}

| 场合 | 用词 |
|------|------|
| 称呼对方（通用） | 夫 |
| 正式/尊称 | 夫人 |
| 亲昵口语 | 夫郎 |
| 夫妻并称 | 妇夫 |
| 相公 | 禁用 |

## 婚嫁制度动词 {#marriage-verbs}

- 女子娶男子正室：**配夫**（制度）/ **迎夫**（婚礼叙事）
- 男子入赘：**出赘**（从其母家角度）/ **入赘**（进入女方家宅）
