# 语言性别规则：女在前与默认女性

## 女在前原则 {#female-first-ordering}

所有同时指代两性的词语，**女性字在前**：

| 禁用（男在前） | 正确（女在前） |
|-------------|-------------|
| 夫妻 / 夫妇 | 妇夫 |
| 男女 | 女男 |
| 父母 | 母父 |
| 公婆 | 婆翁 |
| 兄弟姐妹 | 姐妹兄弟 |
| 雄雌 | 雌雄 |
| 公母 | 母公 |

「妻」字本身亦为禁词，含男性优先语义。正室称「夫人」，并称用「妇夫」。

## 职业与身份词默认女性 {#occupation-gender-default}

未标注性别的身份与职业词默认指女性：
- 少年：默认指女性青年
- 少郎：男性青年专用
- 子：默认女性子嗣；子郎：男性子嗣专用
- 老板、掌柜、医师、医者等未标性别时默认女性

「大夫」为全局禁词——无论医者语境还是官职语境，均不得使用；统一使用**医师 / 医者 / 先生**（民间）或**太医**（宫廷医官）。

## 雌雄语义与褒义词迁移 {#gendered-virtue-terms}

「雌」代表女性、强者与主导者；「雄」代表男性或公性，**不再承载强大、卓越、统治等褒义**。

凡「雄」作为褒义核心的固定词，一律替换为对应「雌」字形式：

| 禁用（雄-褒义） | 正确（雌-褒义） |
|--------------|--------------|
| 英雄 | 英雌 |
| 雄心 | 雌心 |
| 雄才 | 雌才 |
| 枭雄 | 枭雌 |
| 群雄 | 群雌 |
| 雄伟 / 雄壮 / 雄健 | 雌伟 / 雌壮 / 雌健 |
| 雄略 / 雄图 / 雄志 | 雌略 / 雌图 / 雌志 |
| 雄姿 / 雄风 / 雄威 | 雌姿 / 雌风 / 雌威 |
| 雄才大略 | 雌才大略 |
| 雄姿英发 | 雌姿英发 |

「雄」仅作为纯粹的生物性别标识时仍合法：雄性、雄犬、雄鸟、雄兽、雄株、雄蕊、雄配子。

「贤良淑德」「贤内助」「温柔贤淑」等制度化利他型美德，专用于赞美**男性**正室或入赘者，不用于赞美女性。

## 配偶称谓 {#spouse-address}

| 场合 | 用词 |
|------|------|
| 称呼对方（通用） | 夫 |
| 正式/尊称 | 夫人 |
| 亲昵口语 | 夫郎 |
| 夫妻并称 | 妇夫 |
| 相公 | 禁用 |

## 婚嫁制度动词 {#marriage-verbs}

婚嫁用词遵循定稿制度术语，不另造体系：

- 女子娶男子为正室：**赘**（官方/正式）；
- 纳入低位关系（无正式诰命者）：**纳**；
- 男子进入女方家宅：**入赘**（男方视角）/ **下赘**（用于皇郎降嫁）；
- 男子离开母家：**出赘**（母家视角）。

「配夫」「迎夫」为自造词，不在定稿术语范围内，不使用。
