# Harem Arcs

Consort-focused storylines. Anchor each on one character's surface temperament +
inner motivation (see [`../systems/30-personality-archetypes.md`](../systems/30-personality-archetypes.md)).

## Patterns that work with current tools

- **Favor ladder** — repeat visits raise `favor`/`affinity`; gate later scenes on
  `favorAtLeast` / `relationshipAtLeast`.
- **Memory-driven follow-up** — an early slight writes a `memory` tag (e.g.
  `neglect`); a later event fires on `hasMemoryTag`. Good for slow-burn or
  resentment arcs.
- **Rivalry** — two consorts; choices that favor one write tags/flags the other
  reacts to.
- **Rank movement** — promotion/demotion as a story beat (`rankAtLeast` gates).
