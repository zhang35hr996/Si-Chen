## 1. 世界基调

本作世界并非简单的“古代男权社会性别翻转”，而是一个由生理差异、宗教礼法、母系皇权和承养制度共同塑造出的 **女尊帝国** 大妘，国姓为姒。

官方意识形态并不粗暴宣称“男子低贱”，也不采用现代意义上的男女平等话术，而是强调：

> 男女各有天职，内外各循其分。
> 女子承经血，主祭、主政、主军、主产；男子血薄体柔，宜守内、侍奉、承养、育嗣。

因此，这个世界的表层话术是“天职有别”，实际结构则是 **女尊男附**：女子掌握政治、军事、祭祀、生育、财产与公共话语权；男子主要通过婚配、侍奉、承养胎息和后宅地位获得价值。

## 2. 基础生理设定

### 2.1 女子的身体特征

1. **经血神圣**
   女子的经血象征生育能力、天命延续和宗庙合法性。经血不仅是生理现象，也是政治与祭祀资源。

2. **怀胎不虚弱**
   女子怀胎不会造成明显虚弱，也不会天然损害身体。怀胎期间仍可理政、习武、骑射、处理军国大事。

3. **怀胎期间的限制**
   怀胎期间主要需要减少高风险活动，尤其是剧烈纳入式行房。男子在此阶段多承担侍奉、安养、陪伴与礼仪性职责。

4. **长期选择导致女子更强壮**
   历代社会选择、婚配偏好、战争职责和统治职责都倾向于强化女子体能。经过漫长历史演化，女子平均身高、骨密度、肌力、耐力和恢复能力均强于男子。

### 2.2 男子的身体特征

1. **男子血养会体虚**
   男子承担胎息承养时，需要持续以自身血液滋养胎息。长期承养会导致体虚、乏力、气血亏损，严重者寿命受损。

2. **男子不宜外任的生理理由**
   社会认为男子血薄体柔，不适合长途赴任、从军、主祭、治民、巡政、审案和处理高压外务。

3. **审美与分工**
   男子并非一律病弱，而是整体更偏柔美、细腻、宜内养。社会审美强调男子的柔顺、仪态、侍奉能力、承养能力和后宅教养。

## 3. 生育与承养制度

### 3.1 两种生育方式

女子可选择两种生育路径：

#### A. 自孕制

女子自行怀胎十月，直至生产。此种方式不需要男子承养，也不需要所谓“父血”。

特点：

* 血统完全由母系确定。
* 不涉及男子承养资格。
* 对帝王而言，可能具有特殊政治意义，因为可绕开后宫派系。
* 怀胎期间虽不虚弱，但仍会影响侍寝安排、礼仪禁忌和部分高风险行为。

#### B. 承养制

女子怀胎一段时间后，可将胎息转交给自己选择的男子承养。男子以自身血液持续滋养胎息，直至胎息成熟降生。

暂定流程：

1. 女子怀胎至胎息初定。
2. 经医官与祭司确认可转。
3. 举行授养仪式。
4. 女子指定一名男子承养。
5. 男子入册为承养人，并在宫廷或家宅中承担承养责任。
6. 胎息降生后，承养男子或可获得更高后宅地位或封赏。

### 3.2 转胎时间

当前暂定：

> **怀胎三月后，胎息初定，方可转交男子承养。**

理由：

* 一月过早，仪式感与政治重量不足。
* 三月更适合形成“胎息初定”的礼法与医学解释。
* 三月后转胎，女子仍承担孕育开端的神圣性；男子承担后续承养消耗。

### 3.3 承养不可替代原则

承养制的核心规则：

* 胎息一旦指定承养人，不得更换。
* 承养人死亡后，胎息随之断息，通常无法存活。这是非常严重的宫廷事件。
* 若胎息已近足月，承养人死亡后，太医署可立刻催生。

该规则确保男子承养具有不可替代性，也使后宫男子的承养资格成为重要政治资源。

### 3.4 承养责任与法律约束

为确保血脉与胎息安全，国家以法律约束承养人。

暂定律令原则：

1. 女子将胎息授予男子后，该男子即为唯一承养人。
2. 承养人不能将胎息交予他人。
3. 承养人每日需完成规定承养仪程。
4. 因疏忽导致滑胎，承养人需受罚。
5. 故意断养谋害胎息或承养人者，从重论罪。
6. 胎息平安降生后，承养人可获封赏。
7. 承养人死亡后，国家一定会调查。如果确认为自然病亡，承养人本人不追罪; 若此前隐瞒病情，家族受罚, 负责医官、侍从、看守可能被问责。疏忽致死例如：隐瞒体虚， 未按规服药则定为怠养致滑胎，承养人及家族连坐。自尽定为谋杀皇嗣，株连九族。若涉及皇嗣，几乎等同谋逆。

## 5. 皇权与经血祭祀

### 5.1 经血作为统治合法性

女子经血被视为天命、宗嗣与国运的显现。对皇帝而言，不仅证明其仍具生育能力，也证明其仍能承接天命、延续皇统。

因此，皇帝的经血状态是政治事件。

可能设定：

* 每年有经血相关祭祀-传月礼。
* 皇帝经血正常，象征国本稳固。
* 经血异常，可能引发祭司、宗室、朝臣关注。
* 绝经意味着天命将移，皇帝应传位。

### 5.2 绝经与传位

暂定原则：

> 皇帝若确认绝经，应逐步传位于下一代具备经血与继承资格的皇嗣。

但需区分：

* 怀胎期间无经，不视为绝经。
* 产后短期无经，不视为绝经。
* 需连续一段时间无经，才可由祭司与医官确认。

【待定】确认绝经的标准：连续三年，或通过祭司仪式判定。

## 6. 政治结构与社会秩序

### 6.1 历代皇帝

本世界历代皆为女皇帝。皇权长期以母系传承为核心，国家制度、宗法秩序和官僚体系均围绕女性统治建立。

### 6.2 高位权力主要由女子掌握

位高权重者绝大多数为女性，包括：

* 皇帝
* 公主与宗室女性
* 官员
* 将领
* 祭司
* 地方长官
* 朝堂重臣
* 军政核心人物

男子可通过后宅、承养、婚配和侍奉影响权力，但通常不能直接进入正式政治体系。

### 6.3 官方意识形态

国家并不宣称“男子低贱”，而是强调：

> 女子主外，是承血脉、奉天命、护社稷。
> 男子守内，是养血脉、奉宗嗣、全其柔贵。

这种话术将不平等包装为自然秩序、礼法分工与保护性安排。

## 7. 教育与入仕制度

### 7.1 男子可读书，但受教育分流

男子并非完全禁止读书。为保证后宅管理、侍奉礼仪和承养制度正常运行，男子可以学习内学。

男子可学：

* 侍礼
* 内训
* 承养经
* 香药谱
* 房中礼
* 育嗣仪
* 账册法
* 音律、书画、诗词、仪态

### 7.2 男子不得入仕与干政

男子不得担任外朝官职，不得掌军、主祭、治民、理刑、议政。

外学通常被禁止或高度限制，包括：

* 兵法
* 律令
* 财税
* 治水
* 军械
* 官制
* 地方治理
* 国家祭祀核心仪轨

暂定律令：

> 女子主外，承天经、理军国、祭社稷、掌刑名。男子主内，守血脉、奉宗嗣、理内宅、养胎息。男子不得应外试，不得受外官，不得私习兵律政书。

## 8. 皇室亲属与称谓体系

* 皇甥

核心原则：

> 皇室身份与政治重要性主要看女性血源，不看男性承养关系。

* 皇子若被立为太子，其内廷位分应改入东宫体系。
* 皇子成年受封公主后，原皇子正室可随之改称“公主驸”，具体可由册封诏决定。

### 10.4 皇郎配偶称谓

皇郎为皇帝男儿。皇郎成年后可封贵主。

| 身份   | 称号          | 说明                   |
| ---- | ----------- | -------------------- |
| 皇郎正妻 | 皇郎妃          | 娶皇郎或与皇郎成婚的女子，正式称“尚主” |
| 贵主正妻 | 贵主妃         | 皇郎成年封贵主后，其正妻可称“贵主妃”  |
| 皇郎近身承宠者 | 小妾 | 尚未正式列入高位名册者  |

说明：

* 皇郎因其男性身份，一般不纳妾。
* 皇郎婚配多由皇帝赐婚，属于宗室与官员、贵族之间的重要政治联姻。
* 女子尚皇郎，称“皇郎妃”；若皇郎已封贵主，则称“贵主妃”。
* 皇郎所承养或关联的子嗣，是否入宗谱，仍以母系血源与皇帝诏令为准。

### 10.5 婚配动词与礼法用语

本世界正式礼法中，通常不使用“嫁”“娶”作为皇室与官员家宅婚配的核心动词，而根据身份、权力关系与名分等级使用不同用语。

| 场景         | 正式动词             | 说明                  |
| ---------- | ---------------- | ------------------- |
| 皇帝选择新的后宫男子 | 入宫为侍 / 入宫 / 册入内廷 | 男子被选入宫，成为皇帝后宫之人     |
| 皇帝册立正宫     | 封后 / 册立皇后          | 皇帝立正配为皇后            |
| 皇帝晋封后宫男子   | 封 / 晋 / 册        | 如封贵君、晋承徽、册为皇贵君      |
| 皇子选择正室     | 赘                | 男子入皇子府为正室           |
| 皇子选择侧室     | 赘         | 男子入皇子府为侧室           |
| 皇子收近身承宠者   | 纳                | 侧室以下，不称赘，称纳         |
| 官员选择正室     | 赘                | 男子入官员之家为正室          |
| 官员选择侧室     | 赘           | 男子入官员之家为侧室          |
| 官员收低位侍妾    | 纳                | 侧室以下称纳              |
| 皇郎成婚       | 下赘               | 皇郎以男性宗室身份入女方家或入女方婚籍 |
| 女子与皇郎成婚    | 尚皇郎 / 尚主         | 女子尚主皇郎，成为皇郎妃        |
| 女子与贵主成婚    | 尚贵主              | 女子尚贵主，成为贵主妃         |

说明：

* “入宫”用于皇帝选男子入宫，强调男子进入内廷侍奉皇帝。
* “封后”专用于皇帝册立皇后。
* “封、晋、册”用于后宫位分变化。
* “赘”用于皇子、官员、贵族女子选择正室或侧室，表示男子正式入其家宅或府第婚籍。
* “纳”用于侧室以下，表示收为低位承宠者或小宠，不具备正室、侧室的正式礼法地位。
* “下赘”用于皇郎自身成婚，强调皇郎虽为皇室男儿，仍以男子身份入女方婚序。
* “尚主”用于女子与皇郎、贵主成婚，表示其因婚配皇室男儿而获得特殊荣誉身份。
* 所有文本需要避免使用“嫁”“娶”作为核心制度词

示例：

> 陛下选沈氏入宫为侍，初封贵人，后晋承徽。
>
> 大皇子赘陆氏为皇子正室，可称皇子驸。
>
> 大皇子又纳顾氏为近侍，未列侧驸之位。
>
> 林尚书赘沈氏为内卿，后又赘陆氏为侧卿。
>
> 三皇郎下赘崔氏，崔氏尚皇郎，礼部称其为皇郎妃。
>
> 玉章贵主既受封，崔氏改称贵主妃，文书称其尚贵主。

## 11. 游戏时间系统初设

### 11.1 基础时间结构

暂定采用：

> 一年十二个月，每月三个行动日，每日六次基础行动。

即：

* 1年 = 12个月
* 1月 = 上旬 / 中旬 / 下旬
* 每个行动日 = 6次行动
* 1年基础行动次数 = 12 × 3 × 6

### 11.2 每月节奏

| 行动日 | 主题      | 适合事件                   |
| --- | ------- | ---------------------- |
| 上旬  | 朝政 / 外务 | 上朝、奏折、任免、军政、地方事件       |
| 中旬  | 后宫 / 人际 | 召见、侍寝、赏赐、争宠、安抚         |
| 下旬  | 宗嗣 / 内务 | 经血祭祀、怀胎检查、转胎、承养记录、皇嗣培养 |

### 11.3 行动消耗

| 行动类型 |   消耗 | 示例                       |
| ---- | ---: | ------------------------ |
| 轻行动  |   1次 | 朝会、召见、赏赐、批奏折、散步、侍寝      |

### 11.4 生育与时间系统的结合

若采用“三月转胎”设定：

* 自孕完整周期：10个月 = 30个行动日。
* 可转胎节点：怀胎3个月后 = 9个行动日后。
* 承养周期：约7个月 = 21个行动日。

每月下旬可进行：

* 经血状态判定
* 怀胎状态判定
* 胎息稳定度检查
* 承养人健康检查
* 皇嗣成长事件
* 后宫与宗嗣相关随机事件

## 12. 核心游戏循环

玩家作为皇帝或高位女性，需要在三类核心资源之间取舍：

> 江山、后宫、血脉。

### 12.1 江山

包括：

* 朝政
* 民心
* 军事
* 国库收入

### 12.2 后宫

包括：

* 侍君宠爱
* 位分晋降
* 侍寝安排
* 后宫争斗
* 承养子嗣
* 男子情绪与忠诚

### 12.3 血脉

包括：

* 传月祭祀
* 自孕或承养
* 胎息稳定度
* 承养人健康
* 皇嗣出生与培养
* 继承秩序

玩家的每次选择都可能同时影响政治、情感、宗嗣和礼法合法性。

## 13. 叙事冲突来源

本作的长期叙事冲突应同时来自 **外部国家、朝堂财政、后宫关系、宗嗣继承、承养制度与民间舆论**。玩家的选择不应只影响单一数值，而应在江山、后宫与血脉之间形成连锁后果。

### 13.1 与他国联姻


可用剧情包括：

1. 他国请求迎娶皇郎或贵主，以巩固盟约。
2. 他国王子入宫为侍，却不适应大姬后宫礼法。
3. 敌国要求皇子入赘，引发宗室震怒。
4. 边境小国献上美貌男子，实为探子或质子。

### 13.2 后宫斗争

后宫斗争不只是争宠，而是家族、承养资格、皇嗣归属与朝堂派系的延伸。

可用剧情包括：

1. 皇后掌后宫但失宠，贵君受宠却无正宫名分。
2. 高位侍君争夺承养皇嗣的资格。
3. 低位男子因承养能力出众被破格晋封，引发高位不满。
4. 某侍君借母家势力打压异己。
5. 后宫男子因皇帝久不召见而怨气上升。
6. 有人伪造传召、截留赏赐、调换药膳。
7. 承养人被投药、惊吓、陷害，导致胎息不稳。
8. 侍君私下结党，试图影响皇帝立储。
9. 皇后为维持后宫秩序，处置皇帝宠爱的低位男子。
10. 玩家可选择偏宠、制衡、严惩、安抚或放任内斗。

### 13.3 子嗣与继承斗争

皇嗣身份由母系决定，但承养人、后宫位分、宗室支持和礼法都会影响继承局势。

可用剧情包括：

1. 长皇子虽年长，但经血迟迟未至，储位不稳。
2. 幼皇子天资出众，朝臣请求越序立储。
3. 皇郎虽无继承权，却因受宠或联姻牵动宗室派系。
4. 某皇嗣由低位侍君承养，受到高位后宫轻视。
5. 承养人试图借皇嗣身份抬升自身家族。
6. 公主开府后势力坐大，与皇帝发生权力冲突。
7. 宗室长公主支持某一皇子，形成储位集团。
8. 皇帝自孕所出的皇嗣，被认为血统更纯或更具天命。
9. 传月礼、启月礼、承养记录都可能成为继承争议证据。
10. 玩家可选择立长、立贤、立爱，或暂不立储以维持平衡。

### 13.4 民心不稳

百姓仍会因赋税、灾荒、征役而动摇。

可用剧情包括：

1. 皇帝久不行传月礼，坊间出现绝经传闻。
2. 后宫奢靡引发民怨，朝臣请求裁减内廷开支。
3. 民间出现反礼法谣言，宣称男子不应受承养之苦。
4. 宗教势力借经血、胎息、灾异解释煽动人心。
5. 皇室联姻失败、国库空虚、赋税加重，导致地方暴动。
6. 玩家可选择安抚、赈灾、祭祀、镇压、减税或转移矛盾。

### 13.5 国库吃紧

国库压力可来自战争、灾荒、宫廷开支、祭祀、承养医疗、赏赐与地方贪腐。财政紧张应迫使玩家在理想与现实之间取舍。

可用剧情包括：

1. 边境战事消耗军费，内廷开支必须削减。
2. 传月礼耗资巨大，礼部与户部争执。
3. 承养皇嗣需要名贵药材，太医署请求拨银。
4. 后宫晋封、赏赐、修宫导致国库亏空。
5. 地方官贪墨税粮，民心下降。
6. 公主封地拖欠贡赋，中央财政受损。
7. 皇帝若加税，民心下降；若不加税，军费不足。
8. 他国联姻可换取岁贡或贸易，但会牺牲宗室婚配自主。
9. 赈灾、修河、军饷、祭祀、后宫赏赐之间互相争夺预算。
10. 玩家可选择节流、增税、查贪、借债、开矿、联姻换财或削减后宫用度。

### 13.6 承养事故与滑胎

承养制度是宗嗣核心，因此承养人滑胎、失息、血竭或死亡都属于重大政治事件。

可用剧情包括：

1. 承养人因体虚血竭，胎息不稳。
2. 承养人滑胎，太医署需判定是体弱、怠养还是谋害。
3. 承养人被惊吓、投毒、断药，导致滑胎。
4. 高位侍君争夺承养资格，却在承养后身体迅速衰败。
5. 皇帝必须在保承养人性命与保胎息稳定之间选择。
6. 若承养人死亡，胎息可能断养；临产前可尝试催生，但皇嗣多半体弱。
7. 承养事故牵连其家族、举荐者、医官、宫人和后宫派系。
8. 玩家可选择严查、遮掩、重罚、安抚、追封承养人，或将事故转化为政治清洗机会。

### 13.7 朝堂派系与后宫联动

朝臣、宗室、后宫男子及其母家之间存在利益网络。后宫事件往往会外溢为朝堂斗争。

可用剧情包括：

1. 某贵君母家掌兵，皇帝不敢轻易冷落。
2. 皇后家族势大，压制其他侍君晋封。
3. 新入宫男子出身寒微，却因受宠改变朝堂格局。
4. 朝臣借承养事故弹劾后宫管理失职。
5. 武官集团反对异国侍君承养皇嗣。
6. 皇帝若以内宠干政，威望下降。
7. 玩家可利用后宫制衡朝臣，也可能被后宫派系反噬。

### 13.8 个人情感与制度冲突

玩家作为皇帝，并非只面对政治计算，也会面对情感、偏爱、责任与礼法之间的矛盾。

可用剧情包括：

1. 最爱的男子体弱，不适合承养皇嗣。
2. 最合适的承养人并非玩家宠爱之人。
3. 皇后贤明但无情，宠侍温顺却无政治价值。
4. 某男子不愿承养，却因家族压力必须争取资格。
5. 皇帝想自孕，却遭到后宫与朝臣反对。
6. 皇嗣出生后，皇帝更亲近生父式承养人，引发其他侍君不满。
7. 玩家可选择遵循礼法、维护爱情、牺牲个体，或打破惯例。

### 13.9 可复用冲突模板

后续事件设计可按以下模板生成：

1. **外部压力**：他国、边境、贸易、质子、联姻。
2. **内部压力**：朝臣、宗室、官员、祭司、地方。
3. **后宫压力**：争宠、晋封、承养、陷害、嫉恨。
4. **血脉压力**：怀胎、转胎、滑胎、失息、立储。
5. **民生压力**：灾荒、税赋、谣言、暴动、赈灾。
6. **财政压力**：军费、祭祀、宫费、赈灾、封赏。
7. **情感压力**：偏爱、亏欠、牺牲、背叛、保护。

## 14. 总体属性框架
## 属性系统初设

本作属性系统总共分为四组：

1. **国家属性**：玩家作为皇帝需要管理的外部局势。
2. **皇帝属性**：影响统治、祭祀、怀胎、威慑和剧情分支。
3. **侍君属性**：影响宠爱、后宫斗争、承养、忠诚和事件。
4. **皇嗣属性**：影响继承、培养、立储和成年后政治局势。

整体原则是：属性数量保持克制，政治玩法尽量精简，避免影响后宫养成与剧情体验。

---

## 一、国家属性

国家属性不要太多，建议只保留 **5 个明面属性 + 4 个后台属性**。

### 1.1 明面展示给玩家的国家属性

| 属性      | 含义         | 主要影响             |
| ------- | ---------- | ---------------- |
| **国库**  | 朝廷财政储备     | 祭祀、军费、赈灾、赏赐、宫廷建设 |
| **民心**  | 百姓对朝廷的接受度  | 叛乱、谣言、灾荒后果、税收承受力 |
| **军力**  | 军队战力与边境震慑  | 战争、外敌、边境谈判       |
| **朝局**  | 朝臣与官僚体系稳定度 | 政令执行、弹劾、派系斗争     |
| **生产力** | 国家繁荣程度     | 税收               |

这五个属性足够支撑主要政治玩法。

### 1.2 国家后台属性

这些属性不直接显示具体数值，只通过事件体现，适合作为剧情触发器。

| 后台属性     | 作用               |
| -------- | ---------------- |
| **腐败度**  | 影响税收缩水、赈灾失败、地方事件 |
| **边患压力** | 影响外敌入侵、和亲要求、军费需求 |
| **谣言热度** | 影响民心下降、后宫丑闻外泄    |
| **宗室不满** | 影响立储压力、公主争权、谋反   |

---

## 二、皇帝属性

皇帝属性分为 **统治属性** 和 **身体/后宫属性**。

### 2.1 皇帝明面属性

| 属性     | 含义       | 影响             |
| ------ | -------- | -------------- |
| **声望** | 皇帝威严与压制力 | 镇压朝臣、震慑后宫、降低叛乱 |
| **政略** | 理政能力     | 提高朝局、减少政策副作用   |
| **军略** | 军事判断     | 战争、边境、将领信任     |
| **健康** | 天命与生育状态  | 祭祀、怀胎、传位合法性    |

### 2.2 皇帝后台属性

| 属性       | 作用                         |
| -------- | -------------------------- |
| **疑心**   | 影响使用血滴子、处置侍君和官员的倾向         |
| **声望倾向** | 玩家长期表现形成的外界印象，如仁君、暴君、宠君、勤政 |
| **后宫威压** | 后宫男子对皇帝的敬畏程度               |
| **疲劳积累** | 长期高强度理政、侍寝、怀胎造成负面事件        |

## 三、血滴子系统

血滴子系统可用于窥探官员、侍君、皇嗣和宗室，但必须有代价。

### 3.1 可调查对象

* 官员
* 侍君
* 皇嗣
* 宗室

### 3.2 调查收益

| 收益       | 说明             |
| -------- | -------------- |
| **发现秘密** | 私通、贪腐、结党、谋害、隐疾 |
| **提前预警** | 防止滑胎、政变、下毒、叛乱  |
| **获得把柄** | 可用于威胁、废黜、控制    |

### 3.3 调查代价

| 代价         | 说明        |
| ---------- | --------- |
| **忠诚下降**   | 被查者或其派系不满 |
| **恐惧上升**   | 短期稳定，长期疏离 |
| **后宫亲密下降** | 侍君觉得不被信任  |

## 四、侍君属性

侍君属性不要过多，否则养成会变复杂。建议分为 **基础属性、关系属性、后台属性**。

### 4.1 侍君基础属性

| 属性       | 含义         | 影响            |
| -------- | ---------- | ------------- |
| **年龄**   | 年纪阶段       | 承宠、承养风险、性格成熟度 |
| **容貌**   | 外貌吸引力      | 初始宠爱、侍寝吸引     |
| **才情**   | 诗书、音律、谈吐   | 互动事件、宴会表现     |
| **家世**   | 母家背景       | 朝堂派系、晋封起点     |
| **性格**   | 行为倾向       | 事件选择、争宠方式     |
| **健康**   | 身体状态       | 侍寝、承养、病弱事件    |
| **承养资质** | 是否适合承养胎息   | 胎息稳定、滑胎风险     |
| **喜好**   | 礼物、活动、话题偏好 | 提升宠爱和好感       |

### 4.2 侍君关系属性

| 属性     | 含义           | 影响         |
| ------ | ------------ | ---------- |
| **恩宠** | 皇帝当前宠爱程度     | 晋封、侍寝、后宫嫉妒 |
| **忠诚** | 对皇帝和皇权的服从    | 是否告密、背叛、护主 |
| **野心** | 争位、争宠、争承养的欲望 | 后宫斗争频率     |
| **嫉妒** | 对其他侍君的敌意     | 争宠、陷害、冲突   |

其中，**忠诚、野心、嫉妒** 可以部分隐藏，不必全部给玩家看。

### 4.3 侍君后台属性

| 属性       | 作用               |
| -------- | ---------------- |
| **真心**   | 是否真爱皇帝，影响危机时选择   |
| **秘密**   | 隐疾、旧情、外邦身份、母家谋划  |
| **派系绑定** | 与哪一派朝臣、宗室、后宫势力有关 |
| **阴谋倾向** | 越高越容易主动害人或结党     |
| **承压能力** | 被冷落、责罚、血滴子调查后的反应 |

### 4.4 侍君性格建议

| 性格     | 行为倾向           |
| ------ | -------------- |
| **温顺** | 忠诚高，野心低，不易生事   |
| **骄矜** | 家世高，自尊强，受冷落易怨恨 |
| **机敏** | 易察觉阴谋，也可能主动布局  |
| **多情** | 宠爱涨得快，失宠怨恨也涨得快 |
| **清冷** | 宠爱难升，但忠诚稳定     |
| **怯弱** | 容易被欺负，承压能力低    |
| **刚烈** | 不易屈服，受辱可能极端反应  |
| **野心** | 主动争宠、争承养、争位分   |

每个侍君可以有一个主性格和一个副性格。

示例：

* 沈承徽：主性格清冷，副性格刚烈。
* 陆贵君：主性格骄矜，副性格野心。
* 顾美人：主性格温顺，副性格多情。

## 五、皇嗣属性

皇嗣属性也要精简。核心应围绕：

> 能不能继承、值不值得培养、会不会威胁皇帝。

### 5.1 皇嗣明面属性

| 属性     | 含义                | 影响            |
| ------ | ----------------- | ------------- |
| **年龄** | 成长阶段              | 启月礼、封号、培养内容   |
| **身份** | 皇子 / 皇郎 / 公主 / 贵主 | 继承与婚配路径       |
| **健康** | 身体状态              | 夭折、承养后遗症、成年能力 |
| **天赋** | 先天资质              | 学习速度、未来上限     |
| **勤勉** | 努力程度              | 培养收益          |
| **德行** | 仁恕、自律、责任感         | 民心、朝臣支持       |
| **武力** | 武艺与军事能力           | 军方支持、亲征、叛乱    |
| **政才** | 理政能力              | 继承资格、封地治理     |

### 5.2 皇嗣后台属性

| 属性       | 作用                 |
| -------- | ------------------ |
| **野心**   | 是否主动争储、夺权、结党       |
| **亲近度**  | 与皇帝的情感关系           |
| **承养影响** | 承养人对其性格、健康、派系的残留影响 |
| **派系支持** | 朝臣、宗室、后宫支持谁        |
| **黑暗倾向** | 是否容易残忍、背叛、害手足      |
