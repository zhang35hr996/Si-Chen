# 女尊后宫游戏背景设定初版 V0.1

> 文档用途：作为后续剧情、系统、角色、称谓与玩法设计的基础世界观文档。
> 当前状态：核心规则已定；部分术语与制度为暂定，后续可继续细化或替换。

---

## 1. 世界基调

本作世界并非简单的“古代男权社会性别翻转”，而是一个由生理差异、宗教礼法、母系皇权和血养制度共同塑造出的 **礼法女尊帝国**。

官方意识形态并不粗暴宣称“男子低贱”，也不采用现代意义上的男女平等话术，而是强调：

> 男女各有天职，内外各循其分。
> 女子承经血，主祭、主政、主军、主产；男子血薄体柔，宜守内、侍奉、承养、育嗣。

因此，这个世界的表层话术是“天职有别”，实际结构则是 **女尊男附**：女子掌握政治、军事、祭祀、生育、财产与公共话语权；男子主要通过婚配、侍奉、承养胎息和后宅地位获得价值。

## 2. 基础生理设定

### 2.1 女子的身体特征

1. **经血神圣**
   女子的经血象征生育能力、天命延续和宗庙合法性。经血不仅是生理现象，也是政治与祭祀资源。

2. **怀胎不虚弱**
   女子怀胎不会造成明显虚弱，也不会天然损害身体。怀胎期间仍可理政、习武、骑射、处理军国大事。

3. **怀胎期间的限制**
   怀胎期间主要需要减少高风险活动，尤其是剧烈纳入式行房。男子在此阶段多承担侍奉、安养、陪伴与礼仪性职责。

4. **长期选择导致女子更强壮**
   历代社会选择、婚配偏好、战争职责和统治职责都倾向于强化女子体能。经过漫长历史演化，女子平均身高、骨密度、肌力、耐力和恢复能力均强于男子。

### 2.2 男子的身体特征

1. **男子血养会体虚**
   男子承担胎息承养时，需要持续以自身血液滋养胎息。长期承养会导致体虚、乏力、气血亏损，严重者寿命受损。

2. **男子不宜外任的生理理由**
   社会认为男子血薄体柔，不适合长途赴任、从军、主祭、治民、巡政、审案和处理高压外务。

3. **审美与分工**
   男子并非一律病弱，而是整体更偏柔美、细腻、宜内养。社会审美强调男子的柔顺、仪态、侍奉能力、血养能力和后宅教养。

## 3. 生育与承养制度

### 3.1 两种生育方式

女子可选择两种生育路径：

#### A. 自孕制

女子自行怀胎十月，直至生产。此种方式不需要男子血养，也不需要所谓“父血”。

特点：

* 血统完全由母系确定。
* 不涉及男子承养资格。
* 对帝王而言，可能具有特殊政治意义，因为可绕开后宫派系。
* 怀胎期间虽不虚弱，但仍会影响侍寝安排、礼仪禁忌和部分高风险行为。

#### B. 承养制

女子怀胎一段时间后，可将胎息转交给自己选择的男子承养。男子以自身血液持续滋养胎息，直至胎息成熟降生。

暂定流程：

1. 女子怀胎至胎息初定。
2. 经医官与祭司确认可转。
3. 举行转胎/承养仪式。
4. 女子指定一名男子承养。
5. 男子入册为承养人，并在宫廷或家宅中承担承养责任。
6. 胎息降生后，承养男子可获得更高后宅地位或封赏。

### 3.2 转胎时间

当前暂定：

> **怀胎三月后，胎息初定，方可转交男子承养。**

理由：

* 一月过早，仪式感与政治重量不足。
* 三月更适合形成“胎息初定”的礼法与医学解释。
* 三月后转胎，女子仍承担孕育开端的神圣性；男子承担后续承养消耗。

【待定】是否最终采用三月，或改为一月、二月、百日等更具仪式感的节点。

### 3.3 血养不可替代原则

承养制的核心规则：

* 胎息一旦指定承养人，不得更换。
* 不可由其他男子代血。
* 不可由女子代血。
* 不可使用储存血、动物血或他人血液替代。
* 私自转交胎息、断血、怠养或代养均属重罪。

该规则确保男子承养具有不可替代性，也使后宫男子的承养资格成为重要政治资源。

### 3.4 承养责任与法律约束

为确保血脉与胎息安全，国家以法律约束承养人。

暂定律令原则：

1. 女子将胎息交予某男子后，该男子即为唯一承养人。
2. 承养人不得将胎息交予他人。
3. 承养人每日/每旬需完成规定承养仪程。
4. 因疏忽导致胎息死亡，承养人需受罚。
5. 故意断养、私转胎息、谋害胎息者，从重论罪。
6. 胎息平安降生后，承养人可获封赏、地位提升或终身供养。

## 4. 承养相关术语

为避免“血父”等称呼过于直白，并避免父权意味，暂定采用以下术语体系。

| 概念         | 暂定术语 | 用法说明              |
| ---------- | ---- | ----------------- |
| 胎儿/孕育生命    | 胎息   | 带有医学与玄学双重意味       |
| 转交胎息并由男子血养 | 承养   | 正式制度名称            |
| 法律身份       | 承养人  | 律令、案卷、登记用语        |
| 宫廷正式封号     | 承嗣君  | 男子承养皇嗣或贵族子嗣时的正式尊称 |
| 胎息成功降生后的尊称 | 育嗣君  | 可作为晋封或荣誉称号        |
| 日常简称       | 养君   | 后宫或家宅口语称呼         |

示例：

> 陛下怀胎三月，胎息初定，择沈氏承养。沈氏入册为承养人，宫中称承嗣君。若皇嗣平安降生，则晋为育嗣君，终身受奉。

## 5. 皇权与经血祭祀

### 5.1 经血作为统治合法性

女子经血被视为天命、宗嗣与国运的显现。对皇帝而言，经血不仅证明其仍具生育能力，也证明其仍能承接天命、延续皇统。

因此，女帝的经血状态是政治事件。

可能设定：

* 每月或每年有经血相关祭祀。
* 皇帝经血正常，象征国本稳固。
* 经血异常，可能引发祭司、宗室、朝臣关注。
* 绝经意味着天命将移，皇帝应传位。

### 5.2 绝经与传位

暂定原则：

> 皇帝若确认绝经，应逐步传位于下一代具备经血与继承资格的皇嗣。

但需区分：

* 怀胎期间无经，不视为绝经。
* 产后短期无经，不视为绝经。
* 需连续一段时间无经，才可由祭司与医官确认。

【待定】确认绝经的标准：连续一年、三年，或通过祭司仪式判定。

## 6. 政治结构与社会秩序

### 6.1 历代女帝

本世界历代皆为女帝。皇权长期以母系传承为核心，国家制度、宗法秩序和官僚体系均围绕女性统治建立。

### 6.2 高位权力主要由女子掌握

位高权重者绝大多数为女性，包括：

* 皇帝
* 公主与宗室女性
* 女官
* 将领
* 祭司
* 地方长官
* 朝堂重臣
* 军政核心人物

男子可通过后宅、承养、婚配和侍奉影响权力，但通常不能直接进入正式政治体系。

### 6.3 官方意识形态

国家并不宣称“男子低贱”，而是强调：

> 女子主外，是承经血、奉天命、护社稷。
> 男子守内，是养血脉、奉宗嗣、全其柔贵。

这种话术将不平等包装为自然秩序、礼法分工与保护性安排。

## 7. 教育与入仕制度

### 7.1 男子可读书，但受教育分流

男子并非完全禁止读书。为保证后宅管理、侍奉礼仪和承养制度正常运行，男子可以学习内学。

男子可学：

* 侍礼
* 内训
* 血养经
* 胎息录
* 香药谱
* 房中礼
* 育嗣仪
* 账册法
* 音律、书画、诗词、仪态

### 7.2 男子不得入仕与干政

男子不得担任外朝官职，不得掌军、主祭、治民、理刑、议政。

外学通常被禁止或高度限制，包括：

* 兵法
* 律令
* 财税
* 治水
* 军械
* 官制
* 地方治理
* 国家祭祀核心仪轨

暂定律令：

> 女子主外，承天经、理军国、祭社稷、掌刑名。男子主内，守血脉、奉宗嗣、理内宅、养胎息。男子不得应外试，不得受外官，不得私习兵律政书。

## 8. 皇室亲属与称谓体系

本世界有意重新定义部分现实词义，避免“子、儿、主”等词默认男性，强化母系秩序。

### 8.1 皇帝亲生子嗣

| 身份     | 未成年/基础称呼 | 成年后称呼 | 说明                   |
| ------ | -------- | ----- | -------------------- |
| 所有子嗣统称 | 皇嗣       | —     | 法律统称，不分男女            |
| 皇帝女儿   | 皇子       | 公主    | “皇子”法律上可中性，但社会默认多指女儿 |
| 皇帝男儿   | 皇郎       | 贵主    | 男性皇嗣；成年后出嫁或入配称贵主     |

### 8.2 “皇子”与“公主”的重定义

* **皇子**：皇帝所出的正式子嗣，法律上不分男女；但因女儿更具继承、封地和政治主体性，社会语境中默认指女儿。
* **公主**：成年受封的女性皇子，拥有封地、府兵、食邑和一定行政权。此处“公主”更接近“主公”“藩主”，不是柔弱称号。

示例：

> 大皇子成年，受封镇北公主，开府领三郡。

### 8.3 皇郎与贵主

* **皇郎**：皇帝所出的男性子嗣。
* **贵主**：皇郎成年后出嫁或入配后的尊称。

示例：

> 三皇郎成年后，封玉章贵主，下降崔氏。

### 8.4 皇帝姐妹与旁支宗室

| 身份          | 称呼            | 说明               |
| ----------- | ------------- | ---------------- |
| 皇帝姐妹        | 长公主 / 皇姊 / 皇妹 | 成年宗室女性，可有封地与政治地位 |
| 皇帝姐妹所出的子嗣   | 皇侄            | 属于较重要的母系旁支       |
| 皇帝姐妹所出的成年女性 | 宗公主 / 郡公主     | 可视地位受封           |
| 皇帝姐妹所出的成年男性 | 宗郎 / 宗君       | 有宗室礼遇，无核心权力      |

### 8.5 通过皇室男子关联的子嗣

经由皇郎、贵主、宗室男子承养或婚配关系进入宗谱，但母系不属于皇帝或长公主主脉者，可称：

* 皇甥
* 宗甥
* 外宗

这些人地位低于皇子、皇郎、皇侄，通常不列皇位继承核心序列。

核心原则：

> 皇室身份与政治重要性主要看女性血源，不看男性承养关系。

## 11. 游戏时间系统初设

### 11.1 基础时间结构

暂定采用：

> 一年十二个月，每月三个行动日，每日五次基础行动。

即：

* 1年 = 12个月
* 1月 = 上旬 / 中旬 / 下旬
* 每个行动日 = 5次行动
* 1年基础行动次数 = 12 × 3 × 5 = 180次

### 11.2 每月节奏

| 行动日 | 主题      | 适合事件                   |
| --- | ------- | ---------------------- |
| 上旬  | 朝政 / 外务 | 上朝、奏折、任免、军政、地方事件       |
| 中旬  | 后宫 / 人际 | 召见、侍寝、赏赐、争宠、安抚         |
| 下旬  | 宗嗣 / 内务 | 经血祭祀、怀胎检查、转胎、承养记录、皇嗣培养 |

### 11.3 行动消耗

| 行动类型 |   消耗 | 示例                       |
| ---- | ---: | ------------------------ |
| 轻行动  |   1次 | 召见、赏赐、查看记录、批小奏折、散步       |
| 中行动  |   2次 | 侍寝、宫宴、深谈、审问、处理后宫事件       |
| 重行动  | 3次以上 | 大朝会、大祭、转胎仪式、册封、立储、处理血养事故 |

### 11.4 生育与时间系统的结合

若采用“三月转胎”设定：

* 自孕完整周期：10个月 = 30个行动日。
* 可转胎节点：怀胎3个月后 = 9个行动日后。
* 承养周期：约7个月 = 21个行动日。

每月下旬可进行：

* 经血状态判定
* 怀胎状态判定
* 胎息稳定度检查
* 承养人健康检查
* 皇嗣成长事件
* 后宫与宗嗣相关随机事件

## 12. 核心游戏循环

玩家作为女帝或高位女性，需要在三类核心资源之间取舍：

> 江山、后宫、血脉。

### 12.1 江山

包括：

* 朝政
* 军权
* 税赋
* 地方治理
* 官员派系
* 宗室平衡
* 民心与威望

### 12.2 后宫

包括：

* 侍君宠爱

* 侍寝安排
* 后宫争斗
* 家族联姻
* 承养资格竞争

### 12.3 血脉

包括：

玩家的每次选择都可能同时影响政治、情感、宗嗣和礼法合法性。

## 13. 叙事冲突来源

可持续发展的剧情冲突包括：

1. 女帝经血异常，引发传位压力。
2. 女帝选择自孕，绕开后宫承养派系。
3. 某低位侍君承养能力极强，引发高位后宫嫉恨。
4. 承养人因怠养导致胎息不稳，是否从重处罚。
5. 某男子私学兵法律令，被女帝发现。
6. 皇后无宠但掌后宫，贵君有宠但无名分优势。
