# Attribute Action-Flow Tables

玩家行动 ↔ 属性变化的"钉死表"。先定流向,后定数值。本文覆盖四块:
**朝政、生产力、宗室不满、特长**。设计哲学(见 [`70-factions-and-pressure.md`](70-factions-and-pressure.md)):
玩家主玩后宫/人情/子嗣/权力制衡,朝政属性只负责**提供压力与后果**,不做重度模拟。

> 数值方向用 ↑ / ↓ 表示,**幅度留待平衡阶段**。隐藏属性后台 0–100,前台显示文案等级。

## 动作枢纽:奏折 与 宴会

实现前先认清两个动作其实是**容器**,不是单一动作:

- **批奏折 = 分型折子**,同一入口下是不同折子类型,各自只动各自的属性:
  | 折子类型 | 主要影响 |
  |---|---|
  | 兴修水利 / 漕运 | 生产力↑,国库↓ |
  | 减税休养 / 劝农 | 生产力↑,国库短期↓,民心↑ |
  | 任官 / 考课整顿吏治 | 朝政↑(任能臣)或忠心↑(任亲信) |
  | 准奏宗室加封 | 宗室不满↓,宗室/外戚权势↑ |
  | 赈灾 | 灾后生产力恢复,国库↓,民心↑ |
- **办宴会 = 三系统交汇枢纽**:请谁(宗室不满)× 谁献艺(特长)× 排场预算(国库/威望/谣言)。
  宴会必须有内部结构(请客名单 / 献艺人选 / 预算),否则三条连接会坍缩成"点一个按钮"。

行动点:上朝整场 1 AP(会话制,抽 2–3 事务)、批奏折按批 1 AP、宴会 1 AP 但产出最丰。
朝政/生产力是**低频玩家操作 + 高比例自动漂移**;后宫/宴会/对话是 AP 主要去向。

## 1. 朝政(官僚效率,0–100,可见)

旧称"大臣能力"。代表官僚系统效率。

**显示文案**:政令壅塞 / 因循敷衍 / 守成尚可 / 政通人和 / 百官用命

| 类别 | 来源 → 效果 |
|---|---|
| **主动 ≥2** | 任贤(上朝抽到/奏折任官提拔能臣)→ 朝政↑(提亲信则忠心↑、朝政平) |
| | 准奏整顿吏治/考课 → 朝政↑,国库小耗 |
| | 血滴子查办贪腐结党 → 朝政短期↑(代价见血滴子链:恐惧↑/暴戾↑) |
| **自动 ≥2** | 勤政高(常批折)→ 朝政缓慢↑;久不理政 → 朝政↓ |
| | 贪腐度高 → 朝政暗降 |
| | 外戚权势过高 → 朝政效率↓/偏向外戚 |
| | 大臣忠心高 → 朝政稳定(波动小) |
| **后果 ≥2** | 朝政高 → 国库收入↑、生产力缓慢↑、政务/赈灾处理好 |
| | 朝政低 → 奏折信息失真、贪腐↑、民心↓ |
| | 朝政高 → 上朝事务成功率↑ |

> 注意:**"批奏折"本身不直接加朝政**——它走"勤政↑ → 朝政维持不衰"。真正的直接杠杆是任官/考课。

## 2. 生产力(0–100,可见)

| 类别 | 来源 → 效果 |
|---|---|
| **主动 ≥2** | 准奏兴修水利/漕运(上朝大政或奏折)→ 生产力↑,国库↓ |
| | 准奏减税休养/劝农(奏折)→ 生产力↑,国库短期↓,民心↑ |
| | 赈灾 → 灾后生产力恢复 |
| **自动 ≥2** | 朝政高 → 生产力缓慢↑ |
| | 贪腐高 → 生产力暗降 |
| | 灾荒 / 战乱 → 生产力↓ |
| | 民心高 → 生产力基础略↑ |
| **后果 ≥2** | 生产力高 → 国库收入↑、民心基础↑ |
| | 生产力低 → 国库收入↓、灾后恢复慢、民心↓ |
| | 生产力骤降 → 流民/谣言热度风险 |

**显示文案**:田园荒芜 / 民生凋敝 / 仓廪粗足 / 五谷丰登 / 国殷民阜

## 3. 宗室不满(0–100,隐藏 + 半可见提示,越高越危险)

补齐审计里"宗室不满调节方式少"的缺口。

**半可见提示**:宗室和睦无怨 / 宗亲偶有微词 / 宗室上书渐多 / 藩王暗结私党

| 类别 | 来源 → 效果 |
|---|---|
| **主动 ≥2** | 办宴会并邀宗室进宫 → 宗室不满↓(耗国库) |
| | 准奏为宗室加封 → 宗室不满↓ **但 宗室/外戚权势↑**(代价侧,见下) |
| | 立储明确 → 宗室不满↓ |
| | (负向主动:杀宗亲 → 不满↑↑;废嫡立庶 → 不满↑) |
| **自动 ≥2** | 无嗣/皇嗣少 → 宗室不满缓慢↑ |
| | 暴戾高 → 宗室不满↑ |
| | 威望高 → 宗室不满↓ |
| | 宗室出身侍君受宠 → 宗室不满↓、外戚权势↑ |
| **后果 ≥2** | 宗室不满高 → 上书逼宫、拥立旁支(政变事件) |
| | 宗室不满高 + 皇权安全低 → 废帝/宗室谋反触发 |
| | 宗室不满低 → 皇权安全↑ |

> **代价原则**(同"外戚强而忠是支柱、强而叛是毒瘤"):加封降不满,但抬高宗室权势,
> 累积后成为拥立旁支的底气。不可做成免费下调旋钮。

## 4. 特长(talent — 标签,非数值)

才情**降级为具体特长标签**(舞蹈/古筝/琵琶/歌喉/棋/书法/丹青/诗词…),
退出属性流向图,纯风味。**无皇帝偏好字段**(皇帝即玩家,偏好是玩家本人口味,不建模),
**无隐藏匹配分**。

| 类别 | 说明 |
|---|---|
| **赋予来源** | 选秀入宫时按出身/随机设定;(可选)长期承宠或宴会历练解锁副特长 |
| **后果(门控,非数值)** | 宴会献艺:玩家凭自己喜好点选谁献艺 → 基础恩宠↑、皇帝情绪/疲劳恢复;在场宗室/清流好感 |
| | 对话:特长作为话题钩子,解锁专属对话线与好感切入 |

**三标签别撞车**:**特长**=擅长表演什么 / **喜好**=喜欢收什么礼 / **性格**=行为倾向。
皇嗣教育(学问/骑射/品行)走**上书房**(`HeirEducation`),不靠特长。
若日后要"后宫谋士/贤内助",另开 `见识/谋略` 隐藏特质,不要塞回特长。

## 跨系统流向(本表四块如何接回主图)

```
任官/考课 ─┐
血滴子查贪 ─┼─> 朝政 ─> 国库收入↑ / 生产力↑ / 奏折准确度↑
勤政(批折)─┘            └(低)─> 贪腐↑ / 民心↓

水利/减税/赈灾 ─> 生产力 ─> 国库收入↑ / 民心基础↑
              贪腐/灾荒 ─┘(暗降)

办宴会(邀宗室) ─┐
准奏宗室加封    ─┼─> 宗室不满↓ ─> 皇权安全↑
立储 / 威望     ─┘   (加封同时 外戚权势↑ ── 反向压力)
无嗣/暴戾 ─> 宗室不满↑ ─> 逼宫/拥立旁支/废帝

特长(标签) ─> 宴会献艺 / 对话 ─> 恩宠↑ / 皇帝情绪恢复 / 在场宾客好感
```
