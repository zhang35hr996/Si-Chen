# Attribute Catalog — 皇帝 / 侍君 / 皇嗣

按主体列出**有哪些属性、显示文案、接到哪条流向**。这是
[`80-attribute-action-flow.md`](80-attribute-action-flow.md) 的上游(行动表的"名词表")。
国家属性(军力/国库/民心/生产力/朝政/外戚权势 + 隐藏 大臣忠心/贪腐/宗室不满/谣言热度)
见 [`80`](80-attribute-action-flow.md) 与 [`70-factions-and-pressure.md`](70-factions-and-pressure.md)。

## 通则

- 后台 **0–100 数值**,前台显示**文案等级**(不全是数字)。
- 属性分**明面**(玩家直接可见)/ **暗属性**(隐藏)。
  **开发期暗属性全显示**;正式版显示 `？？？`,由**血滴子调查**逐步解锁。
- 通用数值→文案带:0–19 极低 / 20–39 偏低 / 40–60 中等 / 61–80 偏高 / 81–100 极高
  ——但每个属性换成**风味文本**,不要都写"高/中/低"。
- **特长**是标签(枚举),不是数值,不走文案带。
- **党羽倾向**是文本枚举,不是数值。

## 一、皇帝(玩家本人)

皇帝即玩家,**无恩宠/容貌/才情/偏好**等被动或口味字段(口味由玩家自己拿主意)。
本作设定皇帝可**自孕**(`bloodline.pregnancy`),故健康亦关乎自孕。

### 明面

| 属性 | 显示 | 作用 / 流向 |
|---|---|---|
| 年龄 | 数字 | 健康风险、传位压力、万寿节 |
| 生日 | 日期 | 每年万寿节 |
| 健康 | 描述 | 行动点、病倒、寿命、自孕 |
| 勤政 | 描述 | → 威望↑ / 朝政维持 / 疲劳↑(见 80) |
| 威望 | 描述 | → 大臣忠心↑ / 宗室不满↓ / 民心↑ / 皇权安全↑ |
| 武力 | 描述 | 刺杀生还、狩猎、皇嗣武学教育、武将世家好感、军中威望小幅 |
| 政略 | 描述 | 奏折、任官、科举、皇嗣政治教育 |

**显示文案**

| 属性 | 低 | 中 | 高 |
|---|---|---|---|
| 健康 | 体弱多病 | 尚算康健 | 精力充沛 |
| 勤政 | 怠于朝政 | 偶理政务 | 宵衣旰食 |
| 威望 | 威信不足 | 威仪尚存 | 朝野畏服 |
| 武力 | 不善骑射 | 略通武艺 | 骁勇善战 |
| 政略 | 拙于政事 | 守成有余 | 权谋老练 |

### 暗属性

| 属性 | 作用 / 流向 |
|---|---|
| 暴戾 | → 侍君恐惧↑ / 民心↓ / 大臣**真实**忠心↓ / 宗室不满↑ / 皇权安全↓ |
| 疲劳 | → 健康↓ / 病倒概率↑(勤政、血滴子频用会推高) |
| 皇权安全 | → 逼宫 / 政变 / 废帝 / 王朝覆灭(综合外戚·宗室·大臣·威望) |

**显示文案**

| 属性 | 低 | 中 | 高 |
|---|---|---|---|
| 暴戾 | 宽仁克制 | 恩威并施 | 酷烈难测 |
| 疲劳 | 神清气定 | 略显倦怠 | 积劳成疾 |
| 皇权安全 | 危机四伏 | 尚可维持 | 皇权稳固 |

> 关键链:`勤政→威望/朝政/疲劳`;`暴戾→恐惧/民心/真实忠心/宗室不满/皇权安全`;
> `威望→忠心/宗室不满/民心/皇权安全`。暴戾是"表面忠顺↑、真实忠心↓"的双刃。

## 二、侍君

### 明面

| 属性 | 显示 | 作用 / 流向 |
|---|---|---|
| 年龄 | 数字 | 成长、衰老、剧情 |
| 位分 | 名称 | 礼法地位、待遇 → 母家忠心↑ / 外戚权势↑ / 后宫地位↑ |
| 恩宠 | 描述 | → 情意可能↑ / 其他侍君妒意↑ / 谣言热度↑ / 母家期待↑ |
| 健康 | 描述 | 生病、冷宫、承养风险 |
| 家世 | 描述 | → 母家权势 → 外戚权势(支柱/毒瘤,见母家忠心) |
| 性格 | 标签 | 行为倾向、礼数表达(见 [`30`](30-personality-archetypes.md))|
| 特长 | 标签 | 宴会献艺、对话话题(标签,无匹配分,见 80)|
| 容貌 | 描述 | 初见好感、侍寝邀请概率↑、恩宠涨更快、妒意↑;后期被情意/野心/家世/皇嗣覆盖 |
| 住处 | 名称 | 宫殿/宫室归属 |
| 喜好 | 标签 | 投其所好赏赐 → 情意↑/恐惧↓;送厌恶之物 → 情意↓/疏离↑ |

**三标签别撞车**:特长=擅长表演什么 / 喜好=喜欢收什么礼 / 性格=行为倾向。

**显示文案**

| 属性 | 0–19 | 20–39 | 40–60 | 61–80 | 81–100 |
|---|---|---|---|---|---|
| 恩宠 | 久未问津 | 恩宠寥寥 | 平平 | 颇得眷顾 | 盛宠不衰 |
| 健康 | 病骨支离 | 体弱多病 | 尚算康健 | 身体强健 | 精力充沛 |
| 容貌 | 姿容平平 | 清秀可亲 | 眉目端正 | 容色出众 | 国色天香 |
| 家世 | 寒门出身 | — | 清贵之家 | — | 门第显赫 |

### 暗属性

| 属性 | 作用 / 流向 |
|---|---|
| 情意 | 是否真心亲近皇帝(恩宠/投喜好可推高)|
| 恐惧 | → 表面顺从↑ / 情意↓ / 暗害·出逃·背叛风险↑(暴戾推高)|
| 野心 | 是否争宠、夺位、扶持皇嗣 |
| 母家忠心 | 母家是否可靠 |
| 母家权势 | 母家是否能制衡皇权 |

**显示文案**

| 属性 | 低 | 中 | 高 |
|---|---|---|---|
| 情意 | 心冷如冰 | 若即若离 | 情深意重 |
| 恐惧 | 坦然自若 | 心存敬畏 | 战战兢兢 |
| 野心 | 安分守己 | 心有盘算 | 图谋甚大 |
| 母家忠心 | 暗藏异志 | 尚可拉拢 | 忠心可用 |
| 母家权势 | 门庭冷落 | 尚有根基 | 权倾一方 |

> **外戚支柱/毒瘤**:`母家权势高 + 母家忠心高 → 军力/朝政/威望↑、皇权安全↑`;
> `母家权势高 + 母家忠心低 → 皇权安全↓、宫变/逼宫风险↑`。强不等于坏,叛才坏。
> 当前代码:恩宠=`standing.favor`;情意≈`relationships.affinity`、信任≈`trust`;
> 恐惧/野心/母家忠心/母家权势为待落地新字段。

## 三、皇嗣

### 明面

| 属性 | 显示 | 作用 / 流向 |
|---|---|---|
| 年龄 | 数字 | 成长、入学、成年、婚配、封爵 |
| 健康 | 描述 | 夭折、生病、成长稳定 |
| 宠爱 | 描述 | → 教育资源↑ / 对皇帝亲近↑ / 继位支持度↑ **但** 其他皇嗣妒↑ / 承养人野心↑ / 后宫党争↑ |
| 天赋 | 描述 | 学习**上限** |
| 努力 | 描述 | 成长**速度** |
| 道德 | 描述 | 仁厚/残忍 → 继位后统治倾向 |
| 武力 | 描述 | 狩猎、护驾、军事剧情(受皇帝武力教育)|
| 政治(=学问) | 描述 | 处理政务、继位能力;即 `HeirEducation.scholarship`(学问),不另设字段 |
| 嫡/庶 | 标签 | 礼法继承权 |
| 承养人 | 名称 | 后宫派系绑定 |

**显示文案**

| 属性 | 低 | 中 | 高 |
|---|---|---|---|
| 健康 | 体弱多病 | 尚算康健 | 身体强健 |
| 宠爱 | 少得眷顾 | 宠爱平平 | 深受宠爱 |
| 天赋 | 资质驽钝 | 略有悟性 | 天资聪颖 |
| 努力 | 懒散怠学 | 尚知用功 | 勤勉好学 |
| 道德 | 心性凉薄 | 品性尚可 | 仁厚端方 |
| 武力 | 不善骑射 | 略通武艺 | 骁勇过人 |
| 政治(=学问) | 不通政事 | 略懂权衡 | 深谙治术 |

> 天赋/努力是**养成元参数**(上限/速度);政治·武力·道德是**被训练的能力**。
> **政治即学问**,对应 `HeirEducation.scholarship`,不另设字段;martial=武力、virtue=道德,
> 三者由**上书房问功课**提升(已落地)。天赋/努力为待落地元参数。
> 宠爱=`Heir.favor`、嫡=`legitimate`、承养人=`adoptiveFatherId` 已落地。

### 暗属性

| 属性 | 作用 / 流向 |
|---|---|
| 野心 | 是否主动争储 → 夺嫡事件 |
| 对皇帝亲近 | 是否信任、依附皇帝 |
| 党羽倾向 | **文本枚举**,依附哪一派 |
| 继位支持度 | 朝臣·宗室·后宫·外戚综合支持 |

**显示文案**

| 属性 | 低 | 中 | 高 |
|---|---|---|---|
| 野心 | 无意储位 | 心有不甘 | 志在东宫 |
| 对皇帝亲近 | 心有隔阂 | 敬而有距 | 孺慕甚深 |
| 继位支持度 | 根基浅薄 | 略有人望 | 众望所归 |

**党羽倾向(文本,非数值)**:无明显党羽 / 亲近皇后 / 依附承养人 / 受母家扶持 /
得清流称许 / 得武将拥护 / 与宗室往来密切 / 朝臣多有观望 / 暗结外臣。

> 关键链:`宠爱→资源/亲近/支持度↑,但 嫉妒/承养人野心/党争↑`(宠爱有代价,非纯加成);
> `嫡 + 高能力 + 高支持度 → 威望↑ / 宗室不满↓ / 皇权安全↑`;
> `野心 + 党羽倾向 → 夺嫡 → 后宫斗争 → 外戚介入`。
