# 称谓系统权威规范

> **本文件是全仓唯一权威来源。**
> 其余文档如与本文件冲突，须以本文件为准，或删除矛盾内容并改为链接此处。
>
> 称谓结果必须由 runtime 结构化数据和纯函数解析。RAG（知识库）只能用于辅助写作，
> 绝不能覆盖 runtime state、自动授予称呼权限或充当称谓 resolver。

---

## 一、架构原则

### 1.1 单一权威来源

本文件为人工维护时的主规范。各文档的职责：

| 文件 | 职责 |
|------|------|
| 本文件 | 权威规范（主规范） |
| `content/world.json` ranks | runtime rank 数据（id/name/grade/order/selfRefs） |
| `content/lexicon.json` | runtime forbiddenTerms / approvedTerms / rankAddressRules |
| `content/knowledge/*.md` | RAG 写作参考，不决定当前角色权限 |
| 其他 `docs/world/*.md` | 背景说明，应链接本文件而非重复矩阵 |

### 1.2 Runtime 是执行权威

所有具体称谓结果必须来自 runtime context，包括：

- 当前位分
- 亲缘关系
- 是否允许直呼名字 / 昵称 / 兄弟称 / 凤主 / 凤君
- 当前场合 register

RAG 条目只能说明制度知识，**不能自动授予当前说话者任何称谓权限**。

### 1.3 RAG 的边界

RAG 不得：

- 覆盖角色当前位分或亲缘关系
- 自动允许使用凤主、凤君、昵称、哥哥或弟弟
- 自动授予 structured claim 权限
- 充当称谓 resolver

Prompt payload 必须把 runtime 解析后的 `selfRef`、`targetAddress`、`allowedAlternates`、
`forbiddenInContext` 直接提供给模型。

## 二、皇帝称谓

### 2.1 皇帝身份与自称

| 类型 | 称谓 |
|------|------|
| 正式身份名词 | 皇帝 |
| 正式臣礼称呼 | 陛下 |
| 宫廷日常尊称 | 皇上 |
| 皇帝绝对主称 | 朕 |

**皇帝自称规则：**

| 场合 | 自称 |
|------|------|
| 朝堂、诏书、典礼、正式谈话 | 朕 |
| 普通内廷谈话 | 朕 |
| 私下、亲近场合 | 朕 / 我 / 吾 |
| 面对太后行子礼 | 儿臣 |

限制：`朕`是绝对主称。`我`只在私下或情绪卸下时用。`吾`偏古雅，适用于特定私人场合。
皇帝不得自称`孤`、`寡人`（这些是东宫/分封专用）或`本宫`。

### 2.2 对皇帝的合法称谓

以下称谓均合法（按场合区分）：

| 称谓 | 用法 |
|------|------|
| 皇帝 | 身份名词；史官、旁白、系统；太后平时直称 |
| 陛下 | 正式当面臣礼 |
| 皇上 | 内廷日常尊称（宫隶、侍君私下、官员私下） |
| 圣上 | 第三人称庄重敬称 |
| 今上 | 区别先帝的第三人称政论、史论 |
| 万岁 | 朝贺、山呼、礼仪呼号 |
| 圣驾 | 皇帝驾临、行止、车驾（非普通第二人称） |
| 母皇 | 皇嗣正式称皇帝 |
| 凤主 | 皇后或获明确 runtime permission 的亲密侍君私下称皇帝 |
| 凤君 | 获明确 runtime permission 的亲密大臣私下称皇帝 |
| 皇儿 / 吾儿 | 太后亲近时称皇帝 |

**全局禁用：** `万岁爷`（见禁词表）。

### 2.3 凤主

允许：皇后；runtime 明确授予 `fengzhu` 权限的亲密侍君；私下或亲密场景。

禁止：普通侍君；官员；平民；外国使臣；正式朝堂或典礼。

### 2.4 凤君

允许：runtime 明确授予 `fengjun` 权限的亲密大臣；私下或亲密场合。

禁止：普通官员自动使用；朝堂正式奏对；奏疏、诏书；平民；外国使臣；仅凭忠诚或好感值自动开启。

> 凤主和凤君都是对**皇帝**的称呼，绝不能用于皇后。

### 2.5 不同人物称皇帝

| 人物 / 场合 | 称呼 |
|------------|------|
| 朝臣正式奏对 | 陛下 |
| 史官、旁白、系统 | 皇帝 |
| 宫隶日常伺候 | 皇上 或 陛下 |
| 侍君公开正式场合 | 陛下 |
| 侍君私下守礼 | 皇上 |
| 亲密侍君 / 皇后私下 | 凤主 |
| 亲密受信大臣私下 | 凤君 |
| 皇嗣 | 母皇；极亲密时母亲、阿母 |
| 外国使臣首次正式觐见 | 大姒皇帝陛下 |
| 外国使臣后续奏对 | 陛下 |
| 官员私下议论 | 皇上、圣上、今上 |
| 太后平时与正式场合 | 皇帝 |
| 太后亲近时 | 皇儿、吾儿 |
| 平民 | 陛下、皇上 |

## 三、龙凤语义

**全局规则：** `龙`与`凤`都指女性，尤指女帝、女性皇权、女性皇统。不采用"男龙女凤"二分。

- 皇后不得被称为凤后、凤君、凤主、凤体、凤驾、龙君。
- 皇帝正配只称：皇后、中宫、后位、内廷之君。
- 一女一男双胎不称`龙凤胎`，系统字段用 `mixedSexTwins`，玩家显示文本为`子郎双生`。

## 四、皇后、太后和太皇太后

### 4.1 皇后

| 称呼 | 说明 |
|------|------|

| 太后 / 皇太后 | 皇嗣制度性称呼太后 |

### 4.3 太皇太后

称：太皇太后 / 太皇太后殿下。「后」字保留，不因男性而修改。

### 5.2 自称规则要点

- `本宫`：长御及以上对低位者。**不得对皇帝使用。**
- `臣侍`：正四品长御及以上对皇帝/高位（含皇后）。
- `侍 / 侍身`：少使至才人对皇帝/高位。`侍身`适合私下、示弱、请罪、情感表达，不能在正式场景中随机替换`侍`。
- `小侍`：常在至官男子对皇帝/高位。
- `才人`属于`侍/侍身`，不是`小侍`。
- `本宫`最低适用到长御（正四品），少使以下不得用本宫。
- 皇后自称`臣侍`，不再用`臣后`（旧称废止）。
- `本侍`废止，不再使用。
- `官男子`不是 deprecated，仍为正式位分。

### 5.3 殿下

后宫从一品（驸）及以上可因位分称殿下（皇后殿下至驸殿下）。

正二品及以下不因后宫位分称殿下。

皇室身份独立赋予殿下称谓：太后殿下、太皇太后殿下、太子殿下、公主殿下、贵主殿下、皇子殿下、皇郎殿下。

### 5.4 侍君正式称呼其他侍君

- 从一品及以上：位分称号 ＋ 殿下（如"皇贵驸殿下"、"驸殿下"）
- 正二品及以下：封号或姓氏 ＋ 当前位分（如"沈昭仪"、"陆承徽"）
- 无封号：位分或姓氏 ＋ 位分

|------|------|
| 朝堂群臣 | 诸卿、众卿、卿等 |

| 严厉质问 | 姓名，你可知罪 |

### 6.3 称皇嗣

正式：大皇子、三皇子、四皇郎、太子、某公主、某贵主。私下：正名、小名。

### 6.4 称宫隶和平民

对平辈、小辈：名字、职司、你。隐藏身份私访对陌生长辈：老伯、老嫂。

## 九、官职名称

官员职位不得含`夫`（官职字段）或`郎`（官职字段）。

已落实：御史大夫→御史，六部侍郎→六部副尚书，郎中→部司正，员外郎→部司副正。

**不误伤：** 夫人、命夫、侧夫、男郎、皇郎、子郎——这些不是官职字段。

官职字段校验器：name 含`夫`（排除夫人/命夫/侧夫等白名单）或含`郎`（排除男郎/皇郎/子郎等白名单）则报错。

## 十、官员家族与家宅身份

### 10.1 配偶与伴侣

| 身份 | 称谓 |
|------|------|
| 男性正室 | 夫人 |
| 高品官员受封正室 | 命夫 |
| 正式侧室 | 侧夫 |
| 低位伴侣 | 小配 |
| 最低伴侣 | 通房 |

废止旧称：内卿→夫人，命内卿→命夫，侧卿→侧夫，小宠→小配或通房。

### 10.2 官员与正室

- 女性官员称正室：夫人
- 官员对外称：自称是该夫人的妇主
- 正室或命夫对官员：称妇主，自称夫身
- 女性官员在皇室/侍君面前：自称臣

### 10.3 低位侧室、小配、通房自称

对妇主、夫人/命夫、高位侧室：统一自称`小侍`。私下对同位或低位：`我`。

### 10.4 父母和家长

- 官员称女性家长：母亲；男性家长：父亲
- 上一代女性家长的家宅身份：老妇人（在皇室/侍君前自称老身）

### 10.5 子嗣称谓

`子`默认指女孩。

| 类型 | 称法 |
|------|------|
| 女性子嗣（外人） | 大娘子、长娘子、二娘子 |
| 女性子嗣在皇室前 | 臣女 |
| 男性子嗣（制度） | 子郎 |
| 男性子嗣（外人） | 大哥儿、长哥儿、二哥儿 |
| 男性子嗣在皇室前 | 臣男 / 臣郎 |

`子女`为全局禁词。泛称后代用：子嗣、诸儿、儿辈、后代。
